The stack installs itself.

Eight performance and shader mods, pinned to versions verified against each Minecraft release — bundled into the client so you never touch a mods folder.

Shaders work day one — Iris ships with every version, and Origin's Shader Performance Mode halves shadow cost on any pack for a big FPS win you can toggle off any time. Every pin is boot-verified per version; if a mod isn't stable on a version, it doesn't ship there.
